My idea for this project was to use user input, OpenAI, and some kind of other API as an image source to generate a new 'imaginary' album and tracklist based on how the user is feeling.

For the images I am using The Met Museum's open API, and specifically retrieving images and artwork titles.

Gave OpenAI some of my precious moneys to generate an API key, and using the 3.5 model (I couldn't get gpt 4 to work) instructed it to act as an assistant that generates creative titles and tracklists based on how the user enters that they're feeling.

Next, everything broke and I got a million errors in my debug console. Since I was sick and missed some class, Xinxin saved my life and helped me install node.js and get things working!

In my script, the Open AI API returns an album title as well as a tracklist, and slices up the results to display seperately. A random image from The Met's collection is also displayed in a square, as a potential cover for the 'album'
